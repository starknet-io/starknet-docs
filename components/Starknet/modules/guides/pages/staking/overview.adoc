= Staking on Starknet guide overview

[IMPORTANT]
====
This guide is a hands-on tutorial for staking on Starknet. To learn more about Starknet's proof-of-stake consensus mechanism, go to _xref:architecture-and-concepts:staking.adoc[]_.
====

Staking on Starknet is designed to enhance network security and decentralization by allowing users to either as _validtors_, stake their them directly, or as _delegators_, delegating them to other validators.

This guide will walk you through the various procedures enabled as validators and delegators, including:

* xref:staking/changing-operational-address.adoc[Changing operational address]

.Only delegators

* xref:staking/switching-delegation-pools.adoc[Switching delegation pools]

.Validators and delegators
