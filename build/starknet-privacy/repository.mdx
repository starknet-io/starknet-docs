## Source of truth

- **Layout & scripts:** [README](https://github.com/starkware-libs/starknet-privacy/blob/main/README.md) (or default branch) on GitHub.
- **Protocol:** [IACR ePrint 2026/474](https://eprint.iacr.org/2026/474.pdf).

## What to look for after cloning

| Artifact | Typical role |
| -------- | ------------ |
| **`Scarb.toml`** / packages | Cairo contracts (privacy pool, helpers for open notes / swaps). |
| **`snfoundry.toml`** | Starknet Foundry test and deployment config. |
| **Rust workspace** | Stwo-based prover, action compiler, or CLI (`Cargo.toml` at root or in `crates/`). |
| **Scripts / `Makefile`** | Devnet, deploy, or end-to-end demos. |
| **`.github/workflows`** | CI commands you can mirror locally. |

```bash
git clone https://github.com/starkware-libs/starknet-privacy.git
cd starknet-privacy
ls -la
# Open README.md for exact build and run commands
```

When the repository is public, update this page with a short **directory tree** if the maintainers add one to the README.
