Standard AMMs expose amounts through reserve changes, so this flow targets anonymity (identity hiding), not full amount privacy.

## Open-note swap flow

1. Spend source notes.
2. Create empty open note for destination token.
3. Withdraw source token to helper contract.
4. Invoke helper: swap, then deposit destination token into open note.
