## Open these files first

From repository root, these are the concrete files to check in order:

1. `README.md`
2. `Scarb.toml`
3. `snfoundry.toml` (if present)
4. `rust-toolchain.toml` and `Cargo.toml` (if prover/CLI is in repo)
5. `.github/workflows/*.yml` for CI-validated command sequences

## Install toolchain

```bash
git clone https://github.com/starkware-libs/starknet-privacy.git
cd starknet-privacy

# Cairo + Foundry + Devnet
curl --proto '=https' --tlsv1.2 -sSf https://sh.starkup.sh | sh

scarb --version
snforge --version && sncast --version
starknet-devnet --version
```

If the project includes a Rust prover/client:

```bash
rustup --version
cargo --version
```

## Build and test (minimal, non-redundant)

Run only the commands documented in the project README. Typical sequence is:

```bash
scarb build
snforge test
# cargo build --release   # if prover/client crates are present
```

For repository layout details, see [Repository structure](/build/starknet-privacy/repository).
